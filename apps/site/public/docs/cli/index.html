Reference

CLI reference

The O11yFleet CLI lets you manage configurations, agents, and rollouts from the command line. Install with npm install -g @o11yfleet/cli. The command is ofleet.

Installation

npm install -g @o11yfleet/cli

Or with pnpm:

pnpm add -g @o11yfleet/cli

Authentication

ofleet login --email user@example.com --password <password>

Or set O11YFLEET_API_KEY to the deployment bearer secret used for API automation: 

export O11YFLEET_API_KEY="<API_SECRET>"

Commands

config

Manage configurations.

Command Description
ofleet config:list List all configurations
ofleet config:create --name <name> Create a new configuration
ofleet config:show --config-id <id> Get configuration details
ofleet config:upload --config-id <id> --file <path> Upload YAML and create a new version
ofleet config:rollout --config-id <id> Push current version to connected agents

agents

Manage agents/collectors.

Command Description
ofleet agents:list --config-id <id> List all agents
ofleet agents:list --config-id <id> --stats List agents with aggregate stats

token

Manage enrollment tokens.

Command Description
ofleet token:create --config-id <id> --label <label> Create an enrollment token
ofleet token:list --config-id <id> List enrollment tokens

Examples

Upload and roll out a config

# Create config
ofleet config:create --name "prod-ingress"

# Upload YAML
ofleet config:upload --config-id cfg_xxx --file prod-ingress.yaml

# Start rollout
ofleet config:rollout --config-id cfg_xxx

List agents

ofleet agents:list --config-id cfg_xxx
🚧 CLI extensions: Additional flags (filtering, status selection) and rollout control commands are planned.
